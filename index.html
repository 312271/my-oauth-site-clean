Демо авторизации через Яндекс

Это демонстрация OAuth 2.0 авторизации через Яндекс ID.

Здесь будет информация о статусе авторизации...

Что происходит:

  1. Нажимаете "Войти через Яндекс"
  2. Вас перенаправляет на страницу авторизации Яндекса
  3. Вы вводите логин/пароль и даете разрешения
  4. Яндекс возвращает вас обратно с токеном
  5. Токен сохраняется и используется для получения ваших данных
